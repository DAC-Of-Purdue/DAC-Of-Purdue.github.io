--- layout: project title: Robo Rabbit Run Game permalink: /robo-rabbit-run/ header-image: /assets/rabbit-divider.png long-description: Our club was one of four recipients of the Boiler Gold Rush Entertainment Challenge grant to create an interactive installation for Fall 2024 orientation.
